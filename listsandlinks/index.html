Ordered List of Steps

  1. Create a file in your repository folder by typing: gedit foldername.
  2. Typing the correct HTML code.
  3. Validate the code as you typing.
  4. Check your page on web by typing: python -m SimpleHTTPServer.
  5. After you have done that in terminal, open the browser and typing: localhost:8000 to check your website.
  6. Upload your work to the production server by typing: sftp name1234@cmis-hsteacher
  7. Typing cd public_html to open the folder
  8. Typing: put filename to upload your work to the server.
  9. Typing: exit to quit the server and save your works to github by typing: git add filename, git commit -m "Text", git push.
  10. Now you had make a web page on production server.
